Moduł 2, Temat 2: JSON i formularze

Czym jest JSON? Internet nie rozumie skomplikowanych obiektów JavaScriptu z ich funkcjami i metodami. Rozumie za to czysty tekst. JSON (JavaScript Object Notation) to sposób na zamianę naszego obiektu w zwykły ciąg znaków (tekst), tak by móc go przesłać kablem do serwera.

Schemat: Zamiana (Serializacja) obiektu JavaScript na tekstowy format JSON i odwrotnie.

1. Magia JSON.stringify() i JSON.parse()

Aby przetłumaczyć dane w obie strony, JS posiada wbudowany obiekt JSON z dwiema najważniejszymi metodami:

const uzytkownik = { imie: "Anna", wiek: 25 }; // 1. Zmieniamy obiekt JS w tekst JSON (aby wysłać do serwera): const jsonTekst = JSON.stringify(uzytkownik); // Wynik: '{"imie":"Anna","wiek":25}' (Zwróć uwagę na cudzysłowy przy kluczach!) // 2. Odbieramy tekst z serwera i zamieniamy go z powrotem w obiekt JS: const odebranyObiekt = JSON.parse(jsonTekst); // Teraz znowu możemy używać: odebranyObiekt.imie

2. Pobieranie danych z formularza i metoda POST

Kiedy wysyłamy dane na serwer, nasza funkcja fetch() musi zostać odpowiednio skonfigurowana. Domyślnie robi ona "GET" (pobieranie). My musimy jej powiedzieć: "Zrób POST (wysyłanie), tutaj masz moje dane w formacie JSON, i wiedz, że to jest JSON".
